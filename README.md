# Portafolio

## Descripción del proyecto

Este proyecto es una página web de portafolio personal desarrollada como parte del Sprint 4. El sitio presenta información profesional de una desarrolladora Full-Stack Junior, sus habilidades técnicas, proyectos destacados y datos de contacto.

El objetivo principal del proyecto es construir una página responsiva siguiendo un diseño en Figma, aplicando buenas prácticas de HTML, CSS, metodología BEM y organización de archivos.

## Funcionalidad

La página incluye:

- Encabezado con presentación profesional.
- Enlaces de contacto y descarga de CV.
- Sección de habilidades y herramientas.
- Sección de proyectos destacados con enlaces a demo y GitHub.
- Footer con correo electrónico y redes sociales.
- Diseño responsivo para escritorio, tablet y móvil.

## Tecnologías y técnicas utilizadas

- HTML5 semántico.
- CSS3.
- Flexbox.
- CSS Grid.
- Media queries.
- Metodología BEM.
- Fuentes personalizadas con `@font-face`.
- Diseño responsivo.
- Git y GitHub.
- GitHub Pages.
