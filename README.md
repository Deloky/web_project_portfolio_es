# Portafolio | Camilo Linero

## Descripción del proyecto

Este proyecto es mi portafolio personal como Desarrollador Full-Stack Junior. El sitio presenta mi perfil profesional, habilidades técnicas, proyectos destacados y diferentes formas de contacto.

El objetivo principal del portafolio es mostrar mi experiencia práctica en desarrollo web, mi manejo de tecnologías como HTML, CSS, JavaScript, React, Node.js y Python, y mi capacidad para construir interfaces limpias, responsivas y funcionales.

## Funcionalidad

La página incluye:

- Encabezado con presentación profesional.
- Imagen de perfil.
- Enlaces para descargar CV, visitar GitHub y contactar por WhatsApp.
- Sección de habilidades y herramientas.
- Sección de proyectos destacados con enlaces a demo en vivo y repositorio en GitHub.
- Footer con correo electrónico y enlace a LinkedIn.
- Diseño responsivo para escritorio, tablet y móvil.

## Tecnologías y técnicas utilizadas

- HTML5 semántico.
- CSS3.
- Flexbox.
- Media queries.
- Metodología BEM.
- Fuentes personalizadas con `@font-face`.
- `font-display: swap` para mejorar la carga de fuentes.
- Diseño responsivo.
- Git y GitHub.
- GitHub Pages.
- Figma como referencia de diseño.
